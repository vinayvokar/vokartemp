JDBC

follow step to setup JDBC

Requirements

Step 1 : Download JDK ( "ignore if you already download").

Step 2 : Download mySQL ( "ignore if you already download").

Step 3 : Download JDBC connector driver ( "ignore if you already download").


Setup JDK(skip if you already done);

Step 1 : Go to C:\Program Files\Java\jdk-16.0.2\bin

Step 2: Copy path

Step 3: search environment variables and than open

Step 4: click on path >> click on new >> paste copied path

Step 5: click ok

Setup JDBC

Step 1 : Copy downloaded JDBC connector driver folder

Step 2: Go to C:\Program Files\MySQL and than paste

Step 1 : search environment variables and than open

Step 2: click on new

Step 3: enter name as "classpath"

Step 4: click on browse file

Step 5: go to C:\Program Files\MySQL\mysql-connector-java-8.0.26\mysql-connector-java-8.0.26.jar and select

Step 6: add ;.; at the end of path

Step 7: click ok

Step 8: open cmd run command : javap com.mysql.jdbc.Driver
